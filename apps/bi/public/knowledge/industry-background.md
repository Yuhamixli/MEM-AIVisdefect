## 1. 复合材料拉挤是什么生意

- **拉挤**：连续纤维浸胶 → 进加热模具 → 拉出固化 → 得到**等截面**型材（像「挤牙膏」但材料是纤维+树脂）。  
- **典型用途**：风电叶片主梁/辅梁用的拉挤板、轨交与基建型材、电力绝缘件等。  
- **为什么在意表面**：外观与近表面缺陷影响验收、耐候，有时关联结构风险；客户投诉成本高。

## 2. 质检现在怎么做

| 方式 | 现实 | 痛点 |
|------|------|------|
| 人工目检 | 仍很普遍 | 疲劳漏检、标准飘、难 24h 全检、一人多机 |
| 专用视觉设备 | 已有拉挤板外观检测产品 | 贵、定制、算法不透明、中小产线难上 |
| 通用视觉平台 | 海康/奥普特/凌云光/天准等 | 要集成商二次开发；复材纹理难 |

**课题机会**：需求真实（已有人卖设备）；我们做**可演进原型 + 管理/BI/成本透明**，适合 MEM 训练与展示，不幻想三周干翻专用设备商。

---

## 3. 和本课题的关系（一句话）

企业要的是「少漏检、稳节拍、可追溯」；我们三个月先交 **说得清、演示得出、账算得清** 的系统雏形，指标冲 DR≥95% 等门禁。

## 4. 你需要记住的名词

| 词 | 意思 |
|----|------|
| 拉挤 / Pultrusion | 连续成型工艺 |
| 玻纤 / 碳纤 | 常见增强纤维 |
| 脱模布 | 部分板材表面工艺层，也有外观缺陷 |
| DR / FPR | 检出率 / 误检率（我们的核心质量指标） |
| 在线检测 | 跟着产线节拍跑，不能太慢 |
| M币 | 课题内部成本货币（不是加密货币） |
