## 1. 工艺简述

拉挤成型（Pultrusion）是将**连续纤维**（玻纤、碳纤等）经**树脂浸渍**后，通过**加热模具**连续拉出并固化成型的工艺。适合**等截面**型材的大批量、连续生产。

可以记成四段：

```
纱架放纱 → 浸胶 → 模具加热固化 → 牵引拉出 → 定长切割
```

---

## 2. 和表面缺陷的关系

| 工艺环节 | 可能缺陷 | 为什么 |
|----------|----------|--------|
| 浸渍 | 气泡、白斑/干斑、纤维暴露 | 胶没浸好或气体没排净 |
| 模具温度 / 拉挤速度 | 裂纹、橘皮、色泽不均 | 固化不均匀 |
| 脱模 / 牵引 | 划伤 | 机械摩擦、模具表面问题 |
| 辅材（如脱模布） | 褶皱、污渍、跑偏等 | 布面工艺问题（部分产线） |

七类缺陷的统一定义见 [`defect-catalog.md`](./defect-catalog.md)。

---

## 3. 为什么要做视觉检测

- **质量**：漏检可能导致客户退货或结构风险争议  
- **效率**：人工难 24h 全检，标准易飘  
- **追溯**：图像与缺陷记录可进质量台账 / 我们的 BI  

本课题做的是**表面**视觉方案；内部缺陷（超声、X 射线等）不在本期范围。
