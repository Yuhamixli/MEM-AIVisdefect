## 如何更新（给后续同步）

1. 改对应行的「状态 / 决议 / 更新日期」。  
2. 重要决议另写会议纪要：`docs/05-reporting/meetings/YYYY-MM-DD-*.md`，并在决议栏链到纪要。  
3. 跑 `apps/bi` 下 `npm run sync-data`，BI 知识库即可看到新版本。  
4. **不要**因单条口头意见直接改 `requirements.md` 硬范围；范围变更走变更控制。
