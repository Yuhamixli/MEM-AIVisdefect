```

   6.1 登记册维护（R1–R7）

   8.1 BI 指标与看板信息架构
   8.2 检测 UI（后期）与管理看板

   9.3 光学方案
   9.4 系统集成与演示
10. 收尾
   10.1 文档归档 / M币结算 / 演示验收
```
