# 数据集管理计划

**状态**: 草案  
**更新**: 2026-07-11

## 数据采集

| 项 | 计划 |
|----|------|
| 来源 | **项目负责人提供** → `docs/agent-knowledge-base/raw-data/`（企业/实验室/开源可作为补充） |
| 规模 | 初始 ≥ 2000 张（含缺陷与良品） |
| 分辨率 | ≥ 1920×1080 |
| 多样性 | 不同光照、角度、拉挤速度 |

## 数据标注

| 项 | 计划 |
|----|------|
| 工具 | CVAT / LabelImg / Roboflow |
| 格式 | COCO JSON / YOLO TXT / VOC XML |
| 粒度 | 矩形框（检测）+ 可选实例掩码（分割） |
| 质量 | 标注规范 + 10% 人工复核 |

## 版本管理

- 工具候选: DVC / HuggingFace Datasets / 自定义版本表
- 每次实验固定数据集版本号

## 数据增强

- 基础: 翻转、旋转、亮度、噪声
- 高级: Mosaic、MixUp
- 高级可选: Sim2Real 合成缺陷
