## 3. 物距估算（长边 200mm 约束）

设定：长边 200mm 为约束；按最小 5mm 缺陷估算，像素余量较大。

| 传感器 | 物理尺寸 | 比例关系 | **物距 OC** |
|--------|----------|----------|-------------|
| 3200×1920 | 6.4mm × 3.84mm | 6.4 : 200 = 16 : OC | **≈ 500mm** |
| 3072×2084 | 7.37mm × 4.91mm | 7.37 : 200 = 16 : OC | **≈ 434mm** |

演示/详设口径：在分辨率未关单前，可按「500mm 或 434mm 二选一」并列标注，禁止锁死单一数字。

---

## 4. OpenCV 畸变纠正

1. 棋盘格标定板置于拍摄画面中央  
2. 采集 **15–20** 张图  
3. `cv2.calibrateCamera()` 得畸变系数  
4. `cv2.undistort()` 纠正后再进检测/标注  

---

## 5. 光源：先 DOE，不经验锁型

需结合以下维度做实验设计（过程版原文）：

1. 眩光控制  
2. 照明均匀性  
3. 多缺陷覆盖（理想覆盖七类表面缺陷）  
4. OpenCV 前处理后缺陷对比度强、易筛选  
5. 适用于 **4 面**外观检查场景  

→ 对齐未决 **OD-COM-05**：光源未定型前不做型号冻结。
