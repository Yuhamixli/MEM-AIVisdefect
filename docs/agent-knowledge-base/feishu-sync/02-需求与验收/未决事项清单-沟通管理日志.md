仅对高耦合、改变系统形态的分叉做轻量方案对比：本期仅 OD-COM-03（耦合机械传输）与 OD-COM-08（范围分叉）；其余一律填决策卡，不逐项 SWOT。

1.4 纪律
拍板后才改状态，禁止伪造决议：群聊未正式关单的一律 open / discussing（如 D-03 相关项）。
不因单条口头意见改《需求文档》硬范围；范围变更走变更控制。
