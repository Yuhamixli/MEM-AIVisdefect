1. 项目概述
面向新能源汽车电芯压条用复合材料拉挤成型产线，研发表面缺陷机器视觉检测能力，覆盖：图像采集方案（硬件选型+拍摄规范）、缺陷定义卡 + 检测模型 + 离线检测模块（结构化输出，利 SPC/批次追溯）、以及第三方可独立操作的使用说明。对接 APQP 控制计划与特性清单中的外观项；现状七类缺陷完全人工目检，系统输出为初步判断，替代目检初筛、不替代最终放行。

FR-D5

（可选）异常筛查/分割/VLM
