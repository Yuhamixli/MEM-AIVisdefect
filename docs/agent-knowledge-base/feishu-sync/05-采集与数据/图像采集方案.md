1. 对象与约束
检测对象：新能源汽车电芯压条用复合材料拉挤成型件表面外观（对外脱敏：V公司）。
产品尺寸：宽度 30~50mm，长度约 2m。
缺陷尺度：现场典型缺陷约 8~10mm。
精度要求：5mm（加严管控），像元余量按此下限校核，见 §3.3。
缺陷类别：七类（裂纹、气泡、缺纱、划伤、异物、发白、脏污）；本期先三类 = 裂纹、气泡、划伤【建议，待确认，与《七类缺陷定义卡》一致】。
考核口径（任务书 P0）：抽 50 件测试，≥3 类表面缺陷、查全率 ≥80%、准确率 ≥85%。采集方案的定位是「受控成像」——让缺陷在图像中稳定、可复现地显现，是该门禁的先决条件。
工况基线：走-停-走，卷帘快门面阵相机适配；「连续输送 + 全局快门」为备选系统形态【未决@OD-COM-03】。

3. 物距与视野计算（保留原计算过程）
过程版原注：以下计算设定长边 200mm 为约束条件，按照最小 5mm 的缺陷尺寸估算，像素余量很大。
几何关系（相似三角形）：传感器长边 : FOV 长边 = 焦距 : 物距（OC）。
3.1 方案 A：3200×1920（传感器物理尺寸 6.4mm×3.84mm）
6.4 : 200 = 16 : OC
OC = 200 × 16 / 6.4 = 500mm
物距 = 500mm。
短边 FOV = 200 × 1920 / 3200 = 120mm，覆盖 50mm 产品宽度后两侧各约 35mm 余量，可吸收摆放偏差与边缘畸变。
像元精度 = 3200 px / 200mm = 16 px/mm。
3.2 方案 B：3072×2084（传感器物理尺寸 7.37mm×4.91mm）
7.37 : 200 = 16 : OC
OC = 200 × 16 / 7.37 ≈ 434mm
物距 ≈ 434mm。
短边 FOV = 200 × 2084 / 3072 ≈ 135.7mm，宽度余量更充裕。
像元精度 = 3072 px / 200mm ≈ 15.4 px/mm。

注意：余量以「受控对焦 + 合规照度」为前提；运动中拍摄或照度不足时有效余量显著下降，故 §4 的拍摄规范是余量成立的条件。
4. 拍摄规范
4.1 走-停-走节拍配合
拍摄仅在「停」相执行：到位信号 → 触发拍照 → 图像回传完成 → 放行进入下一段。
暂定 2S/张节拍下，单件单面约 10 段（2m / 200mm）；4 面布置见 §6。
单段节拍预算 = 停稳时间 + 曝光时间 + 传输时间【数值待 OD-COM-04 现场参数回填】。
禁止项：未停稳触发（卷帘逐行曝光产生形变）、曝光中振动、环境光直射入镜头、裸手触碰镜头/标定板。
4.2 标定流程（OpenCV，沿用过程版）
将棋盘格标定板放置在拍摄画面中央；
拍 15-20 张图（变换标定板角度与位置，覆盖全画幅）；
cv2.calibrateCamera() 获取相机内参与畸变系数；
cv2.undistort() 纠正后图像再进入标注 / 检测管线。
补充规范【建议】：
标定板规格（格数 / 格距 / 打印精度）随分辨率定版后确定【待补，关联 OD-COM-16】。
复校触发条件：拆装镜头、物距变更、换型、设备碰撞或搬迁后。
尺寸标定（px→mm 换算系数）：以 §3 像元精度为基准值、标定板实测复核，随 OD-COM-16 关单后冻结（承接既有 OD-2.2）。
4.3 换型标定与产品族复用（差异化方向）
产品族复用：电芯压条宽度 30~50mm 属同一产品族；§3 短边 FOV（120 / 135.7mm）对全族覆盖，故换宽度不换硬件、不重标定，仅调整治具定位。
换型动作清单：治具调整 → 首件试拍 → 像素余量复核 → 登记采集台账。
复用主张：「一次标定、产品族复用」写入第三方使用说明，是本项目「第三方可独立操作」交付的组成部分（差异化能力，非额外承诺）。
5. 光源方案【未决@OD-COM-05，必须 DOE】
原则（已决议口径）：不凭经验锁光源；≤2026-07-30 DOE 立项，样本到位后执行，2026-08-15 前冻结光源。
5.1 五维度评价（DOE 评分准则，沿用过程版）
眩光控制：拉挤表面光泽反射不过曝、不产生干扰亮斑；
照明均匀性：画幅内灰度波动可控，分段间亮度一致；
多缺陷覆盖：理想状态可覆盖 7 种表面缺陷（先三类必须覆盖）；
前处理对比度：OpenCV 前处理后缺陷对比度强、易筛选；
4 面检查适用性：光路几何可复用到 4 面外观检查场景。
每维度 1~5 分 × 权重加权，权重在 DOE 立项会上冻结【待补】。
