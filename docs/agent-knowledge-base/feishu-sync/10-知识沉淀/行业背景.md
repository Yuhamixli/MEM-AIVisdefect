1. 宏观背景：新能源车放量带动电池包结构件需求
产量基线：2025年我国新能源汽车产量 1652.4万辆（国家统计局口径，2026-07-12组内调研）。
需求逻辑链：新能源车产量 → 动力电池包装机量 → 电芯压条等电池包结构件需求。电芯压条在电池包内承担压紧/固定电芯、绝缘与结构支撑功能，且对轻量化与一致性敏感，是纤维增强复合材料的典型应用位。
使用纪律：该产量数字只用于刻画行业景气与需求方向，禁止用「新能源车产量×单车用量×单价」连乘推市场容量——该「产量乘法」已在 BP 评审中明确否决（见第4节）。
2. 需求落点：电芯压条与拉挤复材制品
电芯压条为长条形等截面零件，与拉挤工艺「连续等截面、纤维轴向为主、生产效率高」的特长天然匹配，因此拉挤型材成为该零件的主流制造路线之一。本项目案例企业 V公司的电芯压条拉挤产线，产品宽 30~50mm、长约 2m，现场缺陷尺寸约 8~10mm（项目内口径）；其七类表面缺陷（裂纹、气泡、缺纱、划伤、异物、发白、脏污）目前为 100% 人工目检（2026-07-12组内调研口径）——这就是本课题的真实需求原点。
从零件族视角看，电芯压条并非孤例：电池包内还有一批「长条、等截面、批量走量」的结构与功能件（如各类压紧/支撑/防护类型材），与拉挤工艺的特长高度同构；储能柜等相邻场景亦存在同类外观检测需求（通用工艺知识口径，不作为市场数字依据）。这意味着一套「成像规范＋缺陷标准＋检测模型」的资产具备跨零件、跨截面复用的潜力——这是本项目产品族复用策略的需求侧基础（复用打法见《行业难点与我们怎么选战场》）。
需求还沿供应链向下传导：整车厂与电池厂对零部件供应商普遍执行汽车行业质量体系（APQP/PPAP 先期质量策划与生产件批准、SPC 过程控制、批次追溯等通用要求），外观项作为来料与出厂质量的门面指标，其记录的完整性、判定的稳定性正被纳入审核视野。人工目检「判完即走、不留结构化数据」的模式，在这一传导链条上越来越难以自证——这是「机器视觉替代人工目检的初步判断」被提上日程的体系性原因。
3. 人工目检的四个痛点（为什么是现在）
效率低：2m 长件×多面覆盖，人眼逐件全检与产线节拍（线速为米/分钟级，行业通用量级）存在结构性矛盾，检得快就漏得多。
一致性差：允收尺度因人而异、随班次与疲劳漂移，同一道划痕不同质检员可给出不同结论。
无结构化数据：纸质/口头记录无法回答「什么缺陷、在哪件、哪个面、什么时间、哪个批次」，SPC 过程控制与质量追溯缺少数据底座。
APQP 外观项未自动化：汽车行业 APQP/PPAP 质量体系下，外观项仍靠人工判断与手工记录，是质量数字化版图里的空白格。
四个痛点叠在V公司的具体尺度上更显尖锐：缺陷 8~10mm、精度要求 5mm（加严）、每件约 2m 且需多面覆盖——已接近人眼长时间稳定判别的极限区（项目内口径）。痛点即价值锚点：机器视觉的议价逻辑不是「替代人」本身，而是效率（跟住节拍）、一致性（统一允收尺度）、数据（沉淀可追溯的结构化记录）三笔账，这也与本项目「准确度/效率/成本」的评价铁律同构。

5. 市场容量：工位法口径（情景估算，非第三方验证）
TAM 0.96~22.4亿元，基准 5.4亿元；SAM 0.22~5.38亿元，基准 1.47亿元——按目标产线检测工位数×单工位价值量的工位法估算（2026-07-12组内调研/BP 口径）。工位法的基本结构是：可及工位数＝目标行业拉挤产线数×每线可配检测工位数，单工位价值量＝硬件＋软件授权＋集成＋维保年费的打包口径；区间宽度来自「产线数」与「单工位价值量」两个假设的不确定性，因此对外引用必须连带区间与基准值一起讲，禁止只报上沿。
SOM：三年 10/20/33 个付费项目，累计收入 336.6/691.8/1213.2 万元（BP 口径）。
商业模式：硬件外采＋本地软件授权＋集成＋维保；价格带 PoC 5-10万 / 离线站 15-30万 / 在线单线 40-80万 / 维保 8-12%；基准三年毛利率 34.9%，客户单线 ROI 约 24个月（BP 口径，与 PDCA 的 1.5-3年回收期相互印证）。
决策状态：有条件继续——只批准付费 PoC 准备，不承诺全面铺开（BP 口径）。
6. 政策与标准背景
GB 38031—2025《电动汽车用动力蓄电池安全要求》：2025-03-28 发布、2026-07-01 实施（现行，替代 GB 38031—2020）；热扩散要求升级为「不起火、不爆炸」，电池包及其零部件的质量一致性与可追溯压力上行（2026-07-12组内调研口径）。
GB/T 45575—2025《工业产品表面缺陷自动检测系统技术要求》：2025-04-25 发布、2025-11-01 实施（现行），规定了工业产品表面缺陷自动检测系统的架构与组成、系统功能要求、自动检测核心技术、检测流程以及部署与集成技术要求，适用于基于深度神经网络的工业产品表面缺陷自动检测系统的构建与应用——本项目的系统架构、功能与验收设计可直接对齐该标准（标准适用范围据全国标准信息公共服务平台公开页，2026-07-18编辑核；项目内引用口径＝2026-07-12组内调研）。
拉挤型材在线检测国标在起草（2026-07-12组内调研口径）：先发参与者有贡献缺陷定义与评测方法、进而影响标准的机会窗口。
以上标准名称经全国标准信息公共服务平台核对（2026-07-18编辑核）。
