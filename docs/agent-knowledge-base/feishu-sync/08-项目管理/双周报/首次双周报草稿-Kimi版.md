二、本期真实进展（偏已完成，逐项附证据）
1. 开题汇报完成，成绩 B，已完成诚实复盘（已决议 D-07）
7月11-12日课程开课，小组完成开题汇报，指导教师评价为 B，主因创新性不足。复盘结论：不通过堆砌模型名词补强，后续以「应用效果证据」（识别准确度、效率、成本三轴的可验证改善）作为中期叙事主线。证据落点：开题PPT《基于机器视觉的复合材料拉挤制品表面缺陷检测——以V公司为例2.0》、《0711讨论材料_逐字稿与详细总结_合订本》、《项目评价原则》（evaluation-principles.md）。
2. 任务书口径对齐（已决议）
全组已对齐任务书硬交付口径：抽 50 件测试、检出 ≥3 类表面缺陷、查全率 ≥80%、准确率 ≥85%；交付物含图像采集方案、缺陷定义卡+检测模型+离线检测模块（结构化输出）、第三方可独立操作的使用说明。候选缺陷共七类（裂纹、气泡、缺纱、划伤、异物、发白、脏污），先三类建议为裂纹、气泡、划伤（建议，待确认）。证据落点：《任务书》（范汝杰/郭小川两版）、《七类缺陷定义卡正文框架》（defect-catalog-cards.md）。
3. 四层视觉能力栈完成评估并形成 ADR-001（已决议 D-02）
对"四层视觉能力栈"完成技术评估并修正部署形态：在线轻量（L1 PatchCore 异常筛查 + 可选蒸馏 YOLO）→ 近线精修（L2 Grounding DINO 定位 + L3 SAM2 难例分割）→ 离线 VLM 飞轮（L4 预标注→人工确认→训练集升级）。关键结论：禁止四层串成产线每帧主路径——串行召回相乘，两层各 90% 即降至 81%，对 80% 查全率门槛没有余量。证据落点：《ADR-001-four-layer-vision-stack》。
4. 成熟案例与技术路线调研完成初轮
完成行业案例扫描：最接近的直接竞品为西安获德 HD-LJBC（玻纤/碳纤拉挤板双面在线检测），另有 Fraunhofer PulLoop/SURFinloop、ISRA、AMETEK SmartView 等；明确"不能讲国内首创"，真正的对手是人工巡检+自建规则视觉+本地集成商。路线结论：受控成像+轻量监督检测器（YOLO 族）为主线承担验收三类，PatchCore/EfficientAD 并联补漏，DINO/SAM2/VLM 只做工具层。证据落点：《2026-07-12_复合材料拉挤表面缺陷视觉检测_成熟案例与路线分析》。
5. 行业调研与 BP 初版完成
完成行业调研与商业计划书初版：TAM 基准 5.4 亿（区间 0.96~22.4 亿）、SAM 基准 1.47 亿（工位法情景估算，禁用"新能源车产量乘法"）；商业模式=硬件外采+本地软件授权+集成+维保；决策口径=有条件继续（仅批准付费 PoC 准备）。证据落点：《MEM-AIVisdefect_行业调研与商业计划书_20260712》。
6. 图像采集过程版方案成稿（600万卷帘面阵）
完成《过程版_基于600万彩色卷帘快门面阵相机的视觉布置方案》：卷帘快门彩色面阵适配"走-停-走"工况；16mm 定焦；帧率暂定 2S/张；分段拍摄 50mm×200mm；两种候选分辨率（3200×1920 物距 500mm / 3072×2084 物距 434mm）待确认（未决 OD-COM-16）；OpenCV 棋盘格标定流程已写入。光源未定型，需 DOE，不凭经验锁定（未决 OD-COM-05）。
7. 算法环境验证通过
检测模型环境已配置并验证可跑：Python 3.10.11、ultralytics 8.4.90、torch 2.4.1 + torchvision 0.19.1，模型 yolo11s-seg；CPU/GPU 均可运行；依赖走清华镜像。CUDA 版本、GPU 型号、训练超参待补（截图待整理入库）。证据落点：《检测模型环境配置》。标注工具 LabelImg 操作流程同步成稿（conda 环境 huaduo；train/val/test=7:2:1；抽检 10%），证据落点：《图片标注软件与操作》。
