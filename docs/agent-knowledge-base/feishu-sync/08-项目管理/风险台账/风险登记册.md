2. 风险分布与重点
高等级（3 条，周会必过）：R-01 样本/数据、R-02 光源 DOE、R-05 兼职投入。三者均落在 BW1–BW3 窗口，是当前管理主线的直接依据：样本与参数催办@郭小川、DOE 立项@黄雨欣、问卷摸底@范汝杰。
中等级（7 条）：集中在技术叙事与数据质量，多数有明确 OD 单承接，关单即降级。
低等级（1 条）+ 中低（1 条）：R-10/R-12 由财务周检覆盖，不单独占周会议程。
已缓解（1 条）：R-07 由 ADR-001 在架构层规避，转入持续监控（新成员 onboarding 必读 ADR-001）。
3. 升级与联动
触发条件命中→风险 Owner 在周五周报中升级→周六周会判定应对是否加码；
应对失败或跨工作流无法收敛→范汝杰裁决→必要时列入张老师答疑清单（对应《未决清单》§5 升级路径）；
与 OD 清单联动：R-02↔OD-COM-05、R-04↔OD-COM-01、R-06↔OD-COM-13/14、R-08↔OD-COM-16、R-09↔OD-COM-11、R-11↔OD-COM-10；关单后对应风险降一级；
新增风险追加编号 R-13 起，不改历史编号；状态迁移（open→mitigated→closed）须留日期。
4. 评审节奏与职责边界
每周六周会：高等级风险逐条过状态（5 分钟以内），中/低等级只报变化项；触发条件命中的风险自动升为当会议题。
每月末（7/31、8/31、9/30 节点周）：全表重评一次概率/影响，已关单 OD 对应的风险当场降级或关闭，关闭须留日期与依据。
职责边界：风险 Owner 负责监控触发条件与执行应对动作；范汝杰负责登记册维护、跨风险资源裁决与向张老师/课程侧升级；樊晶负责成本类风险的账务证据；任何组员发现新风险可随时提名，经周会确认后入册编号。
5. 未决与假设
概率/影响评级为管理判断，问卷回收与首批数据到位后于 BW2 末重校准一次；
R-01 的默认假设：样本最迟 BW3（8/22 前）部分到位，否则触发结题范围降级预案（建议项，未决@范汝杰）。
