一、叙事逻辑总览（先大→再小→再大）
大（行业）：制造业质量数字化、机器视觉替代人工目检的大趋势。
小（场景）：聚焦复合材料拉挤制品 → 质检分"内部缺陷"与"表面缺陷"两类 → 内部靠 X 光（贵、慢），表面靠人工目检（快、便宜但痛点密集）→ AI 视觉检测切入表面质检，先把"替代人工基本分"拿到。
再大（价值）：从单点检测讲到应用效果证据、商业模式、市场空间与路线图（检测→追溯→分析改进），回到行业尺度收束。
二、硬性写作要求（红线，先读再写）
商业价值必须用应用效果证据说话：效率提升 ≥30%（内部软指标，页面上须标注"软目标/内部口径"）、回收期 1.5~3 年（行业口径）；禁止堆模型名词（不出现"用了某某最新网络所以值钱"式表述；技术只讲"做了什么、效果如何"）。
禁用语："国内首创""无竞争对手"。竞品客观存在（西安获德、Fraunhofer、ISRA、UnitX 及人工巡检/自建规则视觉/本地集成商），竞争页如实呈现并讲差异化。
市场数字只用工位法口径：TAM 基准 5.4 亿（区间 0.96~22.4 亿）、SAM 基准 1.47 亿（区间 0.22~5.38 亿）；区间与情景假设必须标注"情景估算"，禁止用"新能源车产量×单价"乘法外推。
M币与人民币严格分离：M币只用于课程内部核算，一律不进财务模型、不出现在 BP 正文。
决策口径=有条件继续：只批准"付费 PoC 准备"，不承诺量产推广；语气克制，与 D-07"用证据补强"一致。
