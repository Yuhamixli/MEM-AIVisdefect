## 易混淆（待实图补充）

| 对 | 区分提示 |
|----|----------|
| 划伤 vs 裂纹 | 划伤偏机械擦痕；裂纹偏开裂、可能有深度感 |
| 发白 vs 脏污 | 发白多为材质/浸渍外观；脏污为附着污染，可尝试清洁验证 |
| 气泡 vs 异物 | 气泡偏本体空腔/鼓包；异物为外来物边缘更锐或材质不同 |
| 缺纱 vs 发白 | 缺纱见纱束缺失几何；发白以颜色/光泽为主 |

## 与交付分层的关系

| 需求 | 落点 |
|------|------|
| 采集可复现 | 光学方案 + 拍摄规范 |
| 类可定义、可标 | 本图谱 + 定义卡 |
| 可检出、可计量 | 离线检测模块 → 结构化结果（SPC/追溯） |
| 考核 | ≥3 类；50 件；召回≥80%；准确率≥85% |

四层愿景（异常筛查→定位→分割→VLM）仍可作内部技术路线，**本期验收不要求在线产线闭环**。详见 ADR-001。
