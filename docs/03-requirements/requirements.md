## 1. 项目概述

面向新能源汽车**电芯压条**用复合材料**拉挤成型**产线，研发表面缺陷机器视觉检测能力：覆盖**图像采集方案**、**缺陷定义卡 + 检测模型 + 离线检测模块**（结构化输出，利 SPC/批次追溯）、以及**第三方可独立操作的使用说明**。对接 APQP 控制计划与特性清单中的外观项；现状七种缺陷完全人工目检。

| 项 | 内容 |
|----|------|
| 指导老师 | 张老师、朱老师（清华） |
| 团队规模 | 13 人（角色待按能力评估分配） |
| 数据 | 负责人提供，入库 `docs/agent-knowledge-base/raw-data/` |
| 成本货币 | M币（内部核算） |
| 周期 | 开题 2026-07-12 → 结题评审 2026-09-30 |

## 2. 业务目标

## 3. 功能需求

### 3.1 检测核心（P0）

| ID | 需求 | 说明 |
|----|------|------|
| FR-D1 | 缺陷定义卡 | 七类：裂纹、气泡、缺纱、划伤、异物、发白、脏污；含描述/示例/特征/允收 |
| FR-D2 | 缺陷定位与分类 | 中期前至少三类可识别（建议：裂纹、气泡、划伤） |
| FR-D3 | 离线推理 | 封装为可独立运行的离线检测模块 |
| FR-D4 | 结构化输出 | 件号/批次、缺陷类、位置、置信度、时间戳等，利 SPC/追溯 |
| FR-D5 | （可选）异常筛查/分割/VLM | 内部技术路线；不挡申报验收 |

| FR-A3 | 50 件金标准 | 支撑正式评测 |

## 5. 范围边界

### In Scope
- 电芯压条表面七类缺陷的定义、采集、离线检测与文档交付
- 申报考核三项交付物与指标
- 行业/技术调研与结题材料、M币核算、BI 管理看板

### Out of Scope（本期不做，除非变更）
- 完整产线 PLC 深度集成与安全认证（非必须）
- 多工厂 SaaS 化
- 力学/内部缺陷（超声/X射线）
- 以旧 DR≥95% 作为结题唯一硬门禁
