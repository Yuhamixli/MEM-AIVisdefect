# 会议纪要目录约定

**更新**: 2026-07-12  
**完成项**: REPO-TODO `R-A04`

## 命名

```
YYYY-MM-DD-<slug>.md
```

例：`2026-07-11-zhang-first-meeting.md`

附件放 `attachments/`，在纪要内用相对链接引用。

## 必填块（对齐模板）

复制根目录 [`templates/meeting-minutes.md`](../../../templates/meeting-minutes.md)，至少包含：

1. 元数据（主题/日期/出席）  
2. 议程  
3. 讨论要点  
4. **决议**（可执行、有负责人）  
5. **待办**（可进 TODO / open-decisions）  
6. **M币影响**（无则写「无」）  
7. **与评价原则对齐**（是否服务准确度/效率/成本；有无为炫技加复杂度）

## 节奏建议

| 场合 | 产出 |
|------|------|
| 导师沟通 | 本目录纪要 + 必要时更新 `evaluation-principles` / `open-decisions` |
| 组内周会 | 纪要 + 周报（`../daily/`） |
| 里程碑评审 | 纪要 + 阶段报告（`../phase/`） |

见导师 / 写周报前：扫最近 1–2 份本目录纪要 + `docs/feedback-inbox/inbox/`。
